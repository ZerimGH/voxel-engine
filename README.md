# Building
Right now, the Makefile will only work on linux and similar operating systems (No windows or macos support yet)

Dependencies:
  - GLFW
  - GLEW
  - pthread.h (Will be included in POSIX compliant systems)

# Controls
Theres like no gameplay right now, not really worth playing

WASD: Movement

F: Zoom

Shift: Crouch

Ctrl: Sprint

Space: Jump

X: Show debug overlay

Left click: Place a block

Right click: Break a block
